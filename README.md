# Travel_Checklist

## Description
Application web permettant de créer et gérer une checklist pour préparer un voyage (documents, vêtements, accessoires, etc.).  
Ce projet est le **trente-troisième** du défi personnel **100 projets en 2026**.

---

## Objectifs du projet
- Implémenter un CRUD complet
- Gérer des états (coché / non coché)
- Manipuler des tableaux d’objets
- Améliorer l’expérience utilisateur
- Utiliser le stockage local

---

## Plateforme
- Web (navigateur)

---

## Technologies utilisées
- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage

---

## Fonctionnalités
- Ajout d’un élément à la checklist
- Cocher / décocher un élément
- Suppression d’un élément
- Affichage du nombre d’éléments restants
- Sauvegarde automatique des données
- Restauration de la checklist au rechargement

---

## Design & UX
- Liste claire avec cases à cocher
- Feedback visuel (élément barré si complété)
- Barre de progression
- Interface simple et intuitive
- Responsive (mobile et desktop)

---

## Ce que j’ai appris
- Gestion d’état dans une liste dynamique
- Manipulation du DOM avec des événements
- Structuration des données en objets
- Utilisation de LocalStorage
